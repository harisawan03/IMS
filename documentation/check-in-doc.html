IT Inventory Management


Checking Items In


The process for checking an item in is nearly identical to checking an item out. Upon selecting Check-in Item, you will be directed to a page to input the name of the employee checking in the item. Again, be sure to type their name exactly as it appears in the database.





Hitting Confirm will update the inventory within the database and bring you to a confirmation page. Here you will be ensured the checkin was successful and the data affected will be displayed. From here, you will be able to return to the home page.

Be sure not to go back to check-in page. Doing so may cause discrepencies in the database.
