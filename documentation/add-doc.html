Adding Items to Inventory


As noted before, when adding an item you may come across one of two possibilties, the item already exisits in the database or is does not. The system will automatically detect if it exists or not based on the UPC and redirect you accordingly.

If the UPC is not recognized, your only option (as seen on the basic usage intro page) will be to add the item.



If the item does already exist in the inventory and the UPC was not recognized, the scanner may have not properly read the bar code. In this case, return to the homepage and try again. If the problem persists and you have verified the item does already exist in the inventory, see the custom SQL documentation. But be sure you understand SQL before proceeding!
